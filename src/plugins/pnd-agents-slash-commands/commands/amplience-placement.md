# /amplience-placement

Configure Amplience content placements, slot assignments, and page composition for dynamic content delivery.

## Usage

```
/amplience-placement [describe the placement or page configuration needed]
```

## Examples

```
/amplience-placement Configure hero banner slot for the homepage
/amplience-placement Set up product carousel placement for category pages
/amplience-placement Create a promotional grid layout for the sale page
```

## What This Command Does

1. **Analyzes** page layout requirements
2. **Configures** slot definitions
3. **Creates** placement rules
4. **Sets up** content scheduling
5. **Defines** targeting conditions
6. **Generates** delivery configuration

## Output Format

The agent provides:
- Slot configuration JSON
- Placement rule definitions
- Content scheduling setup
- Targeting condition logic
- Delivery API query examples
- React integration code

## Placement Features

- **Slots**: Named content containers
- **Editions**: Scheduled content sets
- **Targeting**: Audience-based content
- **Fallbacks**: Default content rules
- **Preview**: Staging environment setup

## Configuration Options

- Slot naming conventions
- Content type restrictions
- Scheduling windows
- Geographic targeting
- Device-specific content
- A/B testing setup

## Pandora CMS Standards

- Consistent slot naming
- Reusable placement patterns
- Clear scheduling workflows
