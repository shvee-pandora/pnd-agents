# /exec-summary

Transform sprint data, Jira updates, or project information into polished executive summaries for leadership.

## Usage

```
/exec-summary [paste sprint data, Jira summary, or project update]
```

## Examples

```
/exec-summary [paste sprint review notes here]
/exec-summary Create an exec update from Sprint 42 completion data
/exec-summary Summarize the Q4 release progress for the VP meeting
```

## What This Command Does

1. **Analyzes** sprint or project data
2. **Extracts** key accomplishments
3. **Identifies** risks and blockers
4. **Highlights** decisions needed
5. **Formats** for executive audience
6. **Creates** actionable next steps

## Output Format

The agent provides:
- One-page executive summary
- Key accomplishments (3-5 bullets)
- Risks and mitigations
- Decisions required from leadership
- Timeline and milestone status
- Resource needs (if any)
- Next sprint/phase preview

## Summary Sections

- **Status**: Green/Yellow/Red with explanation
- **Highlights**: Major wins and completions
- **Risks**: Issues requiring attention
- **Decisions**: Items needing leadership input
- **Metrics**: Key performance indicators
- **Next Steps**: Upcoming priorities

## Audience Considerations

- Director/VP level readers
- Non-technical language
- Business impact focus
- Clear action items
- Time-sensitive callouts

## Pandora Reporting Standards

- Concise, scannable format
- Data-backed statements
- Clear ownership for actions
- Honest risk assessment
- Forward-looking perspective
